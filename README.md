# Calculator
Simple calculator that uses infix to postfix conversion and postfix evaluation to solve infix expressions.

#### WARNING: IT ONLY SUPPORT SINGLE DIGIT EXPRESSION RIGHT NOW Eg. ((1+2)/3)
